MebelCity

website

github

demo

MebelCity is a full-stack eCommerce website developed using the MERN stack (MongoDB, Express, React, Node). The platform provides users with a seamless shopping experience for high-quality modern furniture items.
It features a robust product catalog, a seamless cart system, and secure Stripe payment integration for a hassle-free shopping experience.
Additionally, administrators can efficiently manage products and orders through the intuitive admin panel, while users enjoy convenient order tracking and access to their order history.

Note: The website is hosted on a free Render server. Please allow 50-60s to spin up.

FoodieConnect

github

demo

FoodieConnect is a user-friendly Android app developed using Flutter, designed to enhance dining experiences by simplifying restaurant discovery.
The app utilizes Google Maps and the Google Places API to display nearby restaurants based on the user's current location. With the integration of Firebase services, users can easily read and write reviews, attach images, add restaurants to favorites and create user profiles.
